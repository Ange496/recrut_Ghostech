🚀 PROCHAINE ÉTAPE : DÉPLOYER SUR RENDER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

👉 Ouvrez le fichier : DEPLOY_NOW.md

📊 Résumé rapide :

1. Allez sur https://render.com
   └─ Sign up avec GitHub

2. New > Web Service
   └─ Sélectionnez "recrut_Ghostech"
   └─ Render détecte render.yaml

3. Ajouter les variables d'env :
   ├─ EMAIL_USER = ghostech92@gmail.com
   ├─ EMAIL_PASS = [Mot de passe app Gmail]
   ├─ RECIPIENT_EMAIL = ghostech92@gmail.com
   └─ PORT = 3000

4. Cliquez "Create Web Service"

5. Attendez 2-3 min → URL générée !

6. Mettez à jour ghostech-recrutement.html
   └─ Remplacez : const API_URL = 'https://ghostech-recrutement-backend.onrender.com';

7. Push sur GitHub
   └─ Render redéploie automatiquement ✨

8. VOILÀ ! Votre formulaire est en ligne 🎉
